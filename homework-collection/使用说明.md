## 🚀 快速开始（3 步骤）

### 步骤 1: 进入项目目录
```bash
cd D:\Proj\claude-demo\homework-collection
```

### 步骤 2: 启动开发服务器
```bash
npm run dev
```

或者双击运行：`start.bat`

### 步骤 3: 访问网站
在浏览器打开：**http://localhost:3000**

## ✨ 主要功能

1. **作业列表展示** - 美观的卡片式布局
2. **实时搜索** - 搜索标题、描述、课程、标签
3. **多维筛选** - 按学期、年份、难度筛选
4. **标签过滤** - 多选标签过滤
5. **详情页面** - 完整的作业信息展示
6. **响应式设计** - 支持桌面、平板、移动端

## 📝 添加自己的作业

编辑文件：`src/data/homeworkData.ts`

在 `homeworkData` 数组中添加您的作业数据。

示例：
```typescript
{
  id: 'hw-001',
  title: '您的作业标题',
  description: '简短描述',
  course: '课程名称',
  semester: '2024春季',
  year: 2024,
  tags: ['标签1', '标签2'],
  difficulty: 'intermediate',
  date: '2024-01-01',
  content: '# 详细内容...',
  githubUrl: 'https://github.com/yourname/project'
}
```

## 🎨 自定义

### 修改网站标题
编辑 `src/components/Header.vue`，修改第 14 行

### 修改配色
编辑 `src/components/Header.vue`，修改第 2 行的渐变色

### 修改端口
编辑 `vite.config.ts`，修改 `server.port`

## 📦 构建和部署

### 构建生产版本
```bash
npm run build
```

### 预览生产版本
```bash
npm run preview
```

### 部署
将 `dist/` 目录部署到：
- GitHub Pages
- Netlify
- Vercel
- 或其他静态网站托管服务
