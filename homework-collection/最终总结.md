## 📝 使用说明

### 启动开发服务器
```bash
cd D:\Proj\claude-demo\homework-collection
npm run dev
```

### 访问网站
打开浏览器访问：**http://localhost:3000**

### 添加自己的作业
编辑 `src/data/homeworkData.ts`，在 `homeworkData` 数组中添加新的作业对象。

### 构建生产版本
```bash
npm run build
```

### 部署
将 `dist/` 目录部署到任何静态网站托管服务。
